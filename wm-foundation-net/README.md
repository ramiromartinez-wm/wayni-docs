# Wayni Foundation Framework (wm-foundation)

## Objetivo

Desarrollar un framework base en .NET Core para estandarizar y acelerar la creación de microservicios. El objetivo es resolver de forma nativa las tareas repetitivas del día a día (como logging estructurado, observabilidad, multitenancy y autenticación) a través de módulos listos para conectar al ciclo de vida de la aplicación.
